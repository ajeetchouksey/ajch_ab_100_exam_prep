## ðŸŽ¯ Quick Reference

### Agent Types

| Type | Use Case | Example |
|------|----------|---------|
| **Task Agent** | Specific workflow automation | Invoice processing automation |
| **Autonomous Agent** | Complex multi-step reasoning | Sales forecasting with insights |
| **Prompt Agent** | Generative AI responses | Content creation assistant |
| **Response Agent** | Simple Q&A interactions | FAQ chatbot |

### Core Microsoft Technologies

| Technology | Purpose | Skill Level |
|------------|---------|-------------|
| **Azure AI Foundry** | Agent development platform | Developer |
| **Copilot Studio** | Low-code agent builder | Business Analyst |
| **Dynamics 365 Copilots** | Prebuilt industry agents | End User |
| **Power Platform** | Integration and workflow automation | Power User |
| **Azure OpenAI** | Enterprise LLM services | Developer |

### Key Frameworks

#### Cloud Adoption Framework Phases
1. **Strategy** - Define business justification
2. **Plan** - Align technical teams
3. **Ready** - Prepare Azure environment
4. **Adopt** - Implement and migrate
5. **Govern** - Control and compliance
6. **Manage** - Optimize operations

#### CLEAR Prompt Engineering Framework
- **C**ontext - Set the scenario
- **L**anguage - Specify tone and style
- **E**xamples - Provide sample outputs
- **A**ctions - Define expected behavior
- **R**esults - Clarify success criteria

#### Responsible AI Principles
1. **Fairness** - Treat all people equitably
2. **Reliability & Safety** - Perform consistently
3. **Privacy & Security** - Protect data
4. **Inclusiveness** - Empower everyone
5. **Transparency** - Be understandable
6. **Accountability** - People are responsible
